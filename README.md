# javablog

#### 项目介绍
初学Java，第一个Java博客，SpringBoot框架+thymeleaf前端模板引擎

#### 软件架构
1) SpringBoot 2.0.5.RELEASE
2) thymeleaf模板引擎
3) mysql数据库
4) BootStrap前端页面

#### 安装教程

1. 官网下载SpringBoot框架
2. 安装thymeleaf依赖
3. 测试连接

#### 使用说明
1. Shift+F10 启动编译
2. 访问:http://localhost:8088/
3. 前台和后台
4. 前台html页面已经摘出来,可以独立迁移到其他博客中
    前台html目录：/static/html
